## <a name="impact-of-a-recall-request"></a><span data-ttu-id="332df-137">Dampak dari permintaan penarikan</span><span class="sxs-lookup"><span data-stu-id="332df-137">Impact of a recall request</span></span>

### <a name="operational-impact"></a><span data-ttu-id="332df-139">Dampak operasional</span><span class="sxs-lookup"><span data-stu-id="332df-139">Operational impact</span></span>

<span data-ttu-id="332df-140">Jika permintaan penarikan ulang disetujui, rekaman persetujuan ditandai sebagai **ditolak**.</span><span class="sxs-lookup"><span data-stu-id="332df-140">If a recall request is approved, the approval record is marked as **Rejected**.</span></span> <span data-ttu-id="332df-141">Status entri akan diubah **Dikembalikan** maupun **Ditolak**, tergantung pada apakah entri waktu atau entri pengeluaran.</span><span class="sxs-lookup"><span data-stu-id="332df-141">The status of the entry is changed to either **Returned** or **Rejected**, depending on whether it's a time entry or an expense entry.</span></span>

<span data-ttu-id="332df-142">Anggota tim proyek dapat melihat entri, mengedit, dan kemudian mengajukan kembali entri, atau menghapus entri seluruhnya.</span><span class="sxs-lookup"><span data-stu-id="332df-142">The project team member can view entries, edit and then resubmit entries, or delete entries entirely.</span></span>

<span data-ttu-id="332df-143">Jika permintaan penarikan ulang ditolak, status entri tetap **disetujui**, dan entri tidak dapat diedit oleh anggota tim proyek atau pemberi izin untuk proyek.</span><span class="sxs-lookup"><span data-stu-id="332df-143">If a recall request is rejected, the status of the entry remains **Approved**, and the entry can't be edited by the project team member or the approver for the project.</span></span>

<span data-ttu-id="332df-145">Jika permintaan penarikan disetujui, aktual yang sesuai untuk biaya dan penjualan diperbarui dengan cara berikut:</span><span class="sxs-lookup"><span data-stu-id="332df-145">If a recall request is approved, the corresponding actuals for cost and sales are updated in the following manner:</span></span>

- <span data-ttu-id="332df-146">Bidang **Status penyesuaian** diperbarui ke **disesuaikan**.</span><span class="sxs-lookup"><span data-stu-id="332df-146">The **Adjustment Status** field is updated to **Adjusted**.</span></span>
- <span data-ttu-id="332df-147">Bidang **Status Penagihan** diperbarui ke **Dibatalkan**.</span><span class="sxs-lookup"><span data-stu-id="332df-147">The **Billing Status** field is updated to **Canceled**.</span></span>

<span data-ttu-id="332df-148">Selanjutnya, entri pembalikan dibuat di dalam tabel aktual.</span><span class="sxs-lookup"><span data-stu-id="332df-148">Next, reversal entries are created in the Actuals table.</span></span> <span data-ttu-id="332df-149">Untuk membuat entri pembalikan, sistem menyalin nilai bidang dari aktual asli.</span><span class="sxs-lookup"><span data-stu-id="332df-149">To create reversal entries, the system copies over the field values from the original actuals.</span></span> <span data-ttu-id="332df-150">Hanya nilai yang tidak disalin adalah nilai kuantitas.</span><span class="sxs-lookup"><span data-stu-id="332df-150">The only values that aren't copied over are the quantity values.</span></span> <span data-ttu-id="332df-151">Nilai ini dibalik sebagai gantinya.</span><span class="sxs-lookup"><span data-stu-id="332df-151">These values are reversed instead.</span></span> <span data-ttu-id="332df-152">Aktual terbalik dibuat untuk aktual **biaya** dan **penjualan tidak ditagih**.</span><span class="sxs-lookup"><span data-stu-id="332df-152">Reversed actuals are created for both **Cost** and **Unbilled Sales** actuals.</span></span> <span data-ttu-id="332df-153">Bidang **status penyesuaian** pada aktual terbalik diatur ke **tidak dapat disesuaikan**, dan **status penagihan** diatur ke **dibatalkan**.</span><span class="sxs-lookup"><span data-stu-id="332df-153">The **Adjustment Status** field on the reversed actuals is set to **Unadjustable**, and the **Billing status** field is set to **Canceled**.</span></span> <span data-ttu-id="332df-154">Karena perubahan ini, pengeluaran yang tercatat dan akumulasi pendapatan pada proyek tidak akan mempertimbangkan lagi jumlah yang diwakili oleh aktual ini.</span><span class="sxs-lookup"><span data-stu-id="332df-154">Because of these changes, the recorded spending and the revenue backlog on the project will no longer account for the amounts that these actuals represent.</span></span>

<span data-ttu-id="332df-155">Jika permintaan penarikan ulang ditolak, tidak ada dampak keuangan pada proyek.</span><span class="sxs-lookup"><span data-stu-id="332df-155">If a recall request is rejected, there is no financial impact on the project.</span></span>
