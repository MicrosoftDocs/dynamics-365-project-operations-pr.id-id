# <a name="configure-accounting-for-billable-projects"></a><span data-ttu-id="8c2cb-103">Mengkonfigurasi akuntansi untuk proyek yang bisa ditagih</span><span class="sxs-lookup"><span data-stu-id="8c2cb-103">Configure accounting for billable projects</span></span>
