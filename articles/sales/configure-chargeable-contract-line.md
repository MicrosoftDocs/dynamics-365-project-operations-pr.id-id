# <a name="configure-chargeable-components-of-a-project-contract-line"></a><span data-ttu-id="b5eda-103">Mengonfigurasikan komponen yang dikenakan biaya di baris kontrak proyek</span><span class="sxs-lookup"><span data-stu-id="b5eda-103">Configure chargeable components of a project contract line</span></span>

<span data-ttu-id="b5eda-104">_**Berlaku untuk:** Project Operations untuk skenario berbasis sumber daya/tanpa stok_</span><span class="sxs-lookup"><span data-stu-id="b5eda-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="b5eda-105">Baris kontrak berbasis proyek memiliki konsep komponen yang dikenakan dan tidak dikenakan biaya yang disertakan.</span><span class="sxs-lookup"><span data-stu-id="b5eda-105">A project-based contract line has the concept of included, chargeable, and non-chargeable components.</span></span>

<span data-ttu-id="b5eda-106">Komponen yang disertakan diatur dalam metode penagihan, pemisahan pelanggan, batas yang tidak boleh dilebihi, dan pengaturan frekuensi faktur yang ditentukan pada baris kontrak berbasis proyek.</span><span class="sxs-lookup"><span data-stu-id="b5eda-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based contract line.</span></span>

<span data-ttu-id="b5eda-107">Subkumpulan komponen yang disertakan dapat ditandai sebagai dikenakan biaya dengan memperbarui bidang **Tipe Penagihan**.</span><span class="sxs-lookup"><span data-stu-id="b5eda-107">A subset of the included components can be marked as chargeable by updating the **Billing Type** field.</span></span>

<span data-ttu-id="b5eda-108">Komponen yang dapat dikenakan biaya dapat didefinisikan pada kategori peran dan transaksi.</span><span class="sxs-lookup"><span data-stu-id="b5eda-108">Chargeable components can be defined on roles, and transaction categories.</span></span>

<span data-ttu-id="b5eda-109">Untuk baris kontrak proyek, Bisa dikenakan biaya yang didefinisikan pada peran hanya berlaku untuk kelas transaksi **waktu**.</span><span class="sxs-lookup"><span data-stu-id="b5eda-109">For a project contract line, the chargeability defined on a role only applies to the **Time** transaction class.</span></span> <span data-ttu-id="b5eda-110">Jika **Sertakan waktu** diatur ke **Tidak** di baris kontrak proyek, tab **Peran yang dapat dikenakan biaya** tidak tersedia.</span><span class="sxs-lookup"><span data-stu-id="b5eda-110">If **Include Time** is set to **No** on a project contract line, the **Chargeable roles** tab isn't available.</span></span>

<span data-ttu-id="b5eda-111">Bisa dikenakan biaya yang didefinisikan pada kategori transaksi untuk baris kontrak proyek hanya berlaku untuk kelas transaksi **Pengeluaran**.</span><span class="sxs-lookup"><span data-stu-id="b5eda-111">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="b5eda-112">Jika **Sertakan Pengeluaran** diatur ke **Tidak** di baris kontrak proyek, tab **Kategori yang dapat dikenakan biaya** tidak tersedia.</span><span class="sxs-lookup"><span data-stu-id="b5eda-112">If **Include Expenses** is set to **No** on a project contract line, the **Chargeable Categories** tab isn't available.</span></span>

<span data-ttu-id="b5eda-115">Di tab **Peran kena biaya** pada baris kontrak berbasis proyek, di sub kisi **Kategori kena biaya** pada bidang **Jenis penagihan**, perbarui jenis penagihan untuk peran.</span><span class="sxs-lookup"><span data-stu-id="b5eda-115">On the **Chargeable roles** tab of a projec-based contract line, on the **Chargeable Categories** subgrid, in the **Billing Type** field, update the billing type for a role.</span></span>

### <a name="update-a-transaction-category-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="b5eda-116">Memperbarui kategori transaksi sebagai dikenakan biaya atau tidak dikenakan biaya</span><span class="sxs-lookup"><span data-stu-id="b5eda-116">Update a transaction category to be chargeable or non-chargeable</span></span>

<span data-ttu-id="b5eda-117">Kategori transaksi dapat dikenakan biaya atau tidak dikenakan biaya pada baris kontrak berbasis proyek tertentu.</span><span class="sxs-lookup"><span data-stu-id="b5eda-117">A transaction category can be chargeable or non-chargeable on a specific project-based contract line.</span></span>

<span data-ttu-id="b5eda-118">Di tab **Kategori kena biaya** pada baris kontrak berbasis proyek, di sub kisi **Kategori kena biaya** pada bidang **Jenis penagihan**, perbarui jenis penagihan untuk transaksi.</span><span class="sxs-lookup"><span data-stu-id="b5eda-118">On the **Chargeable Categories** tab of a project-based contract line, on the **Chargeable Categories** subgrid, in the **Billing Type** field, update the billing type for a transaction.</span></span>
