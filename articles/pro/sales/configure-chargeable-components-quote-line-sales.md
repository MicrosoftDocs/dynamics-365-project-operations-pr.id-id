<span data-ttu-id="99174-106">Komponen yang tergantung pada:</span><span class="sxs-lookup"><span data-stu-id="99174-106">Included components are subject to:</span></span>

  - <span data-ttu-id="99174-107">Metode penagihan dan pemisahan pelanggan</span><span class="sxs-lookup"><span data-stu-id="99174-107">Billing method and customer splits</span></span>
  - <span data-ttu-id="99174-108">Batas Jangan terlampaui</span><span class="sxs-lookup"><span data-stu-id="99174-108">Not-to-exceed limits</span></span> 
  - <span data-ttu-id="99174-109">Pengaturan frekuensi faktur yang didefinisikan pada baris kuotasi berbasis proyek</span><span class="sxs-lookup"><span data-stu-id="99174-109">Invoice frequency settings defined on the project-based quote line</span></span>

  - <span data-ttu-id="99174-113">Tidak Dikenakan Biaya</span><span class="sxs-lookup"><span data-stu-id="99174-113">Non-chargeable</span></span>

<span data-ttu-id="99174-114">Komponen yang dapat dikenakan biaya dapat didefinisikan pada kategori tugas, peran, dan transaksi.</span><span class="sxs-lookup"><span data-stu-id="99174-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="99174-117">Bisa dikenakan biaya didefinisikan pada peran untuk baris kuotasi dan hanya berlaku untuk kelas transaksi **waktu**.</span><span class="sxs-lookup"><span data-stu-id="99174-117">Chargeability is defined on roles for a quote line and only applies to the **Time** transaction class.</span></span> <span data-ttu-id="99174-118">Jika bidang, **Sertakan waktu** diatur ke **Tidak** di baris kuotasi proyek, tab **Peran yang dapat dikenakan biaya** tidak tersedia.</span><span class="sxs-lookup"><span data-stu-id="99174-118">If the field, **Include Time** is set to **No** on a project quote line, the **Chargeable Roles** tab isn't available.</span></span>

<span data-ttu-id="99174-119">Bisa dikenakan biaya didefinisikan pada kategori transaksi untuk baris kuotasi dan hanya berlaku untuk kelas transaksi **Pengeluaran**.</span><span class="sxs-lookup"><span data-stu-id="99174-119">Chargeability is defined on transaction categories for a  quote line and only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="99174-120">Jika bidang, **Sertakan Pengeluaran** diatur ke **Tidak** di baris kuotasi proyek, tab **Kategori yang dapat dikenakan biaya** tidak tersedia.</span><span class="sxs-lookup"><span data-stu-id="99174-120">If the field, **Include Expenses** is set to **No** on a project quote line, the **Chargeable Categories** tab isn't available.</span></span>

<span data-ttu-id="99174-122">Tugas proyek dapat dikenakan biaya atau tidak dikenakan biaya dalam konteks baris kuotasi berbasis proyek tertentu yang memungkinkan pengaturan berikut:</span><span class="sxs-lookup"><span data-stu-id="99174-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>

<span data-ttu-id="99174-126">Tipe tagihan tugas dapat dikonfigurasi pada tab **Tugas yang Dikenakan Biaya** dari baris kuotasi berbasis proyek dengan memperbarui bidang **Tipe Penagihan** pada subkisi **tugas baris kuotasi**.</span><span class="sxs-lookup"><span data-stu-id="99174-126">A task's billing type can be configured on the **Chargeable Tasks** tab of a project-based quote line by updating the **Billing Type** field on the **Quote Line Tasks** sub-grid.</span></span> <span data-ttu-id="99174-127">Atau, Anda dapat memperbarui tipe penagihan untuk tugas proyek di bidang **Tipe Penagihan** pada subkisi di konfigurasi Penagihan tugas proyek yang memperlihatkan baris kuotasi yang terkait dengan tugas.</span><span class="sxs-lookup"><span data-stu-id="99174-127">Alternatively, you can update the billing type for a project task in the **Billing Type** field on the sub-grid on the task billing setup of a project that shows the quote lines associated to a task.</span></span>
